Privacy Policy - Chat Outline

Overview

This extension is a local, client-side developer tool designed to display your own ChatGPT or Grok user messages in a convenient in-page overlay. It does not collect, store, transmit, or share personal data.

Data Collection

  • The extension observes network requests and responses made by ChatGPT only on https://chatgpt.com and Grok on https://grok.com.
  • It extracts user-authored message text only from ChatGPT and Grok conversation APIs.
  • No other websites, APIs, or network traffic are monitored.

Data Usage

  • All data is processed in-memory within your browser session.
  • Data is used solely to display user messages in the on-page monitor UI.
  • Data is cleared automatically when a conversation is reset, deleted, or the page is refreshed.

Data Storage

The extension does not use:

  • chrome.storage
  • localStorage
  • sessionStorage
  • IndexedDB

No data is persisted across page reloads or browser sessions.

Data Sharing

  • No data is sent to external servers.
  • No analytics, tracking, or telemetry is used.
  • No data is shared with third parties.

Permissions

  • The extension only requires access to https://chatgpt.com/* and https://grok.com/* to function.
  • No additional permissions (such as storage, cookies, or background access) are requested.

Security

  • All logic runs locally in the browser.
  • The extension does not modify ChatGPT and Grok requests or responses.
  • Native browser behavior is preserved.

Disclaimer

This extension is an independent developer tool and is not affiliated with, endorsed by, or associated with OpenAI, ChatGPT or xAI.
